# **Порядок организации дополнительных занятий**

Для полной прозрачности и понимания со стороны родителей, дополнительные занятия с учеником должны быть четко структурированы: иметь понятные цели, конкретные сроки и измеримый результат.

Занятия проходят в двух основных форматах, каждый из которых имеет свой регламент.

***
### **1. Занятия для ликвидации отставания**

Если ученику рекомендованы дополнительные занятия для устранения пробелов в знаниях, педагог обязан предоставить родителю четкий план.

**Алгоритм действий:**

1.  **Формулировка целей.**
    Педагог составляет и согласовывает с родителем список конкретных тем, над которыми будет вестись работа.

2.  **Определение срока.**
    Указать количество занятий, которые необходимы для проработки тем.

3.  **Предоставление отчета.**
    По итогам курса педагог предоставляет родителю финальный отчет, который содержит:
    *   Перечень пройденных тем.
    *   Заключение о результате: **«Отставание ликвидировано. Уровень знаний ученика соответствует требованиям программы»**.

---

### **2. Занятия для повышенного уровня**

Если ученику рекомендованы занятия для углубленного изучения предмета или развития сверх программы, план также должен быть максимально прозрачным.

**Алгоритм действий:**

1.  **Формулировка целей.**
    Педагог составляет и согласовывает с родителем список тем или навыков для углубленного изучения.

2.  **Определение срока.**
    Указать количество занятий, которые необходимы для проработки тем.

3.  **Предоставление отчета.**
    По итогам курса педагог предоставляет родителю финальный отчет, который содержит:
    *   Перечень освоенных тем и навыков.
    *   Заключение о результате, например: **«Ученик освоил сложные темы на углубленном уровне»**, **«Подготовлен к участию в олимпиаде»** или **«Знания превышают стандартную школьную программу»**.

---
**Главный принцип:** Мы стремимся к полной ясности. Родители должны всегда понимать, над чем работает педагог, сколько это займет времени и какой результат будет достигнут.
