
# Наталья Игоревна
Есть тенденция перехода детей из частных школ в общеобразовательные.
Вышла методичка Академии по финансовому погружению. Необходимо дальше работать в направлении разработки методических пособий, выпускать их и предлагать на их базе обучение для других педагогов.
Для дневничков 1-4 классов можно предлагать свои идеи по наполнению.
Мы получили признание от руководителей других школ.
Закладываем ценности, воспитываем на уроках.
Важна педагогическая этика:
Учитель - ученик (про других учителей, другие школы, что класс трудный)
Учитель - учитель (про других учителей, другие школы).
Курс на сохранение детей в Академии.
Ликвидировать мелкие замечания по предметам (ДЗ, тетради подписаны, активность в чате).
Ждём интегрирование предметов в этом году.
